== Description ==
This is the plugin which would help authors to "Add Books" instead of "Add Posts". It also adds the book on BlogPage. Just install and start writing!


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/booktops` directory, or install the plugin through the WordPress plugins screen directly.
